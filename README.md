# USAAds Premium (Demo)

A simple HTML/CSS/JS demo: watch ads and earn money (simulated, no backend).

## Features
- Premium look
- Watch video ad, earn $0.10 per view
- Balance updates instantly

## How to run
Just open `index.html` in your browser:

```sh
xdg-open index.html  # Linux
# or
open index.html      # macOS
# or
start index.html     # Windows
```

## Notes
- This is a static demo. No real money, no login, no backend.
- For real projects, add authentication, backend, payment, and security.
